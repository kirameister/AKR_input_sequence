# 前置シフト面の配置

単打面の配置ができたところで、シフト面を考えていくことになる。

基本的な考え方は、

* uni-gram のランクが高い拗音ではない文字に対して (ランクが高い順に)、
* tri-gram と bi-gram の表を見て、指の動きが楽な流れになるような配置をする。

現状できている単打面:

```
ー と し 、 つ      く じ こ が は
の シ か ん っ      て う い シ シ
す せ ま る を      た で に 。 な
```

...と上では言ったものの、まずは拗音について考えていく必要がある。これはシフトキーがそのまま拗音文字入力キーになっているためである。作戦としては、bi-gram で拗音文字があるペアの上位から、まだ配置されていない文字を探してきて、それを tri-gram と bi-gram の表を見ながら配置を考えるというものである。

* `き`: `きょう` や `きゅう` といった流れが多い (`きゃ` はあまりない)。
    * 確定: `/lk/` => `き`
* `ち`: `ちょう` や `ちゅう` といった流れが多い。
    * 確定: `/lj/` => `ち`
* `り`: `りょう` や `りょく` といった流れが多い。
    * 確定: `/sj/` => `り`
* `ぎ`: `ぎょう` がとにかく多い。それから周りに `[じうき]` も多い。
    * 確定: `/su/` => `ぎ`
* `ひ`: `ひょ` もあるが、`ひと` や `ひゃく` がある (ようやく `ゃ` がでてきた)。`ひん` もあるので sf あたりで手を打つ。
    * 確定: `/sf/` => `ひ`

現状このような配置になる。

```
__ -- __ __ __      __ ぎ __ __ __      __ __ __ __ __      __ __ __ -- __
__ 〓 __ ひ __      __ り __ == ==      __ == __ __ __      __ ち き 〓 ==
-- -- __ __ __      __ __ __ __ __      __ __ __ __ __      __ __ __ -- --

                    ー と し 、 つ      く じ こ が は
                    の シ か ん っ      て う い シ シ
                    す せ ま る を      た で に 。 な
```
