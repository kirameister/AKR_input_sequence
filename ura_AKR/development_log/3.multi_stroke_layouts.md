# 前置シフト面の配置

単打面の配置ができたところで、シフト面を考えていくことになる。

基本的な考え方は、

* uni-gram のランクが高い拗音ではない文字に対して (ランクが高い順に)、
* tri-gram と bi-gram の表を見て、指の動きが楽な流れになるような配置をする。

現状できている単打面:

```
ー と し 、 つ      く じ こ が は
の シ か ん っ      て う い シ シ
す せ ま る を      た で に 。 な
```

...と上では言ったものの、まずは拗音について考えていく必要がある。これはシフトキーがそのまま拗音文字入力キーになっているためである。
